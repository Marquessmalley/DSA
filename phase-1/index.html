1. What Is Binary Search?

Binary search is a searching algorithm that works on sorted arrays.

It repeatedly divides the search space in half until the target is found (or proven not to exist).

The key insight:

  • â†’ Each comparison eliminates half the remaining elements
  • â†’ This leads to O(log n) time complexity
  • â†’ Vastly faster than linear search for large datasets

2. The Cost of Speed (Important Trade-off)

5. Binary Search Code (Iterative)

def binary_search(arr, target):
    left = 0
    right = len(arr) - 1

    while left <= right:
        mid = (left + right) // 2

        if arr[mid] == target:
            return True
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return False
