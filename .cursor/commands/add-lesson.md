# Add DSA Lesson

Add a new lesson to the DSA course following the established workflow.

## Instructions

1. Read the skill file at `.cursor/skills/add-dsa-lesson/SKILL.md` and follow it step by step.
2. Reference `course-roadmap.md` to identify the lesson details (phase, number, title, topics).
3. Complete every item in the skill's checklist before finishing.

## Input

Provide the lesson details after the command, for example:

```
/add-lesson Phase 2 Lesson 11 Queues
```

If no details are given, check `course-roadmap.md` for the next unimplemented lesson.
